Elementos de pintura

Cambie el tamaño del navegador para ver diferentes versiones de la carga de imágenes en diferentes tamaños de ventana gráfica. El navegador busca el primer elemento fuente donde la consulta de medios coincide con el ancho de la ventana gráfica actual del usuario, y obtiene la imagen especificada en el atributo srcset.

El elemento img se requiere como la última etiqueta secundaria del bloque de declaración de imagen. El elemento img se usa para proporcionar compatibilidad con versiones anteriores para navegadores que no admiten el elemento de imagen, o si ninguna de las etiquetas de origen coincide.

Nota:El elemento de imagen no es compatible con IE12 y versiones anteriores o Safari 9.0 y versiones anteriores.
